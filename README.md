Job Portal Assignment:
----------------------------------------------------------
This is a simple Angular job portal application

Features
--------------------------------
1.Mocked login with username and password
2.Access token and refresh token stored in localStorage
3.Job list loaded from a mocked service
4.Search jobs by title and filter by location
5.Apply for a job using a reactive form
6.Job title and company are auto-filled in the apply form
7.Basic form validation and success message
8.Client-side state management using BehaviorSubject

Tech Stack:
--------------------------------------------
1.Angular
2.TypeScript
3.RxJS
4.Bootstrap


Run Project
--------------------------------------------
npm install
ng serve
